Admin – dohledání podle e‑mailu

Tahle stránka nikam nic neposílá. Jen spočítá stejný SHA‑256 hash e‑mailu jako hra, abys mohl ve Supabase dohledat, kdo je za nickem.

Otevřít hru
Hash (SHA‑256)
—
SQL filtr (pro Supabase → SQL Editor)
—
Postup v Supabase:
  1. Otevři Table Editor → tabulka vajicka_scores
  2. Vyfiltruj sloupec email_hash na hodnotu z boxu výše
  3. Uvidíš záznam(y) s nickem a časem

Reset žebříčku

Z bezpečnostních důvodů reset nejde dělat jedním kliknutím přímo z webu (mohl by to zneužít kdokoliv s odkazem). Níže si zkopíruj SQL a spusť ho v Supabase → SQL Editor.
delete from public.vajicka_scores;
Pozor: nevratně smaže všechny školní výsledky.
